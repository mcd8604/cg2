Billiards Simulation by Mike DeMauro

1. Running the application:
	
	Run the solution using Microsoft Visual Studio 2008

	- Press Space to reset world

2. Building the application:
	
	Build the solution using Microsoft Visual Studio 2008

3. Platform

	Microsoft Windows XP or Vista

4. Notes

	- Linear motion w/ friction supported.
	- Euler integration implemented.
	- No file input implemented.
